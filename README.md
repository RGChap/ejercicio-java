# Gestión de estudiantes en Java

Este proyecto es una aplicación de consola realizada en Java como práctica académica.

Permite gestionar información de estudiantes utilizando arrays y ArrayList.

## Funcionalidades
- Añadir estudiantes
- Mostrar estudiantes
- Buscar estudiantes por nombre
- Eliminar estudiantes por nombre (ArrayList)
- Menú principal y submenús

## Estructura del proyecto
- Clase Estudiante
- Clase Principal
